# Personal Portfolio Website

A simple portfolio website built using **HTML5** and **CSS3**. This project serves as a beginner-friendly web development project that showcases basic website structure, navigation, and styling.

The website consists of two pages:

- **Home Page** – Introduces the portfolio owner and provides a brief overview.
- **Contact Page** – Displays contact information and allows visitors to connect.

## Preview

This portfolio website provides a clean and minimal design that is easy to navigate and suitable for learning the fundamentals of web development.
